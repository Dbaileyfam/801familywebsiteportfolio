# 801 Family Studios — Web Portfolio

An interactive showcase of band websites and electronic press kits built by 801 Family Studios.

## Live preview locally

```bash
cd "website portfolio"
python3 -m http.server 8080
```

Open [http://localhost:8080](http://localhost:8080).

## Deploy to GitHub Pages

1. Push this folder to [801familywebsiteportfolio](https://github.com/Dbaileyfam/801familywebsiteportfolio).
2. On GitHub: **Settings → Pages → Build and deployment → Deploy from branch**.
3. Branch: `main`, folder: `/` (root).
4. Your site will be at `https://dbaileyfam.github.io/801familywebsiteportfolio/`.
